
# Final Fantasy dating app

  Final Fantasy dating app using [MoogleApi](https://github.com/jackfperryjr/mog)

## How to install

  
```
- cd server

- npm install

- cd ..

- cd client

- npm install
```

  

- Copy *.env.local* to /server

## How to run

```
npm run dev
```
in /server and /client folders.

Open http://localhost:5173/ in your browser.
